MATLAB 矩阵（二）

概述

本次的内容有点多，划一下重点：

  • size 函数：创建矩阵和确定循环结构的循环次数
  • reshape 函数：重新排列矩阵
  • sort 函数：内置的 sort 函数比自己编写的排序更高效
  • [] 中括号运算符：串联矩阵最重要的步骤
  • repmat 函数：重复矩阵的拼接

  • 索引：功能强大，针对性的选取数组元素
